## Quatre options de l'assurance de base CSS

###### L'assurance de base CSS est disponible en quatre versions différentes. Le modèle standard est l'assurance maladie obligatoire avec libre choix du médecin. Si vous souhaitez réduire la prime d'assurance, vous pouvez choisir l'un des trois modèles d'assurance suivants:

### Avec le modèle du médecin de famille, les assurés communiquent d'abord avec leur médecin de famille. Si nécessaire, votre médecin vous orientera vers un spécialiste.

### Avec le modèle HMO / GVP, vous consultez un médecin de famille affilié à un cabinet de groupe (HMO) ou à un réseau de médecins si vous avez besoin de traitement.

### Avec le modèle de télémédecine Telmed, vous contactez la hotline gratuite du centre de conseil en télémédecine Medgate en cas de maladie. Vous recevrez les conseils d'un médecin spécialiste qui vous dirigera vers un médecin ou un hôpital si vous avez besoin d'un traitement complémentaire.

## Les assurances complémentaires CSS

###### En plus de l'assurance de base, la CSS propose différentes assurances complémentaires qui vous permettent d'étendre votre couverture d'assurance en fonction de vos besoins. Il s'agit notamment de l'assurance dentaire, de la couverture à l'étranger et de trois modèles myFlex:

### Assurance ambulatoire myFlex : L'assurance complémentaire couvre les frais de traitement ambulatoire dans le monde entier, contribue à la chirurgie oculaire au laser et comprend une assurance dentaire pour les enfants.

### Assurance hospitalisation myFlex : L'assurance hospitalisation complémentaire vous permet de choisir librement l'hôpital et le service hospitalier.

### Assurance alternative myFlex : L'assurance complémentaire couvre les traitements de médecine alternative par des thérapeutes et méthodes agréés par la CSS.

###### Les assurances complémentaires de la CSS comprennent également l'assurance frais médicaux en cas d'accident, l'assurance indemnités journalières d'hospitalisation, l'assurance indemnités journalières individuelles et l'assurance soins dentaires. L'assurance décès ou invalidité en cas d'accident ou de maladie vous permet d'assurer votre famille en cas d'urgence. Si vous fermez le compte de la santé, vous recevrez des subventions pour des mesures de promotion de la santé et de prévention.
